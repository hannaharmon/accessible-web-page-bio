Accessibility Interests

  • Gaming accessibility - Making games playable by everyone through adaptive controls, screen reader support, visual alternatives for audio cues, configurable difficulty, color palettes, etc.
  • Interactive web experiences - Building web apps and interactive tools that work well with assistive technologies and keyboard navigation.
  • Creative tools - Art apps, music makers, design tools. Making creative software accessible means more people can make things.
